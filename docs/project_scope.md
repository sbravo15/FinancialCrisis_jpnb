# Project Scope

## Objective
Build a comparable signal-based narrative for major financial crises instead of a single-case notebook.

## Narrative Framework

1. Pre-crisis build-up
2. Cracks appear
3. Systemic break
4. Real-economy spillover

## Analysis Unit

- Primary level: monthly indicator panel
- Optional high-frequency overlays: daily stress indicators around break events

## Crisis Coverage Plan

- Global Financial Crisis (2008) - baseline implementation
- Dot-com unwind (2000-2002)
- Asian Financial Crisis (1997-1998)
- Savings and Loan crisis (1980s/early 1990s)

## Output Standard

For each crisis:

- phase-aligned timeline chart
- indicator dashboard (housing/credit/markets/labor/production/sentiment)
- short interpretation notes tied to events
